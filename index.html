Counter-Strike 2 Skins Database

This is my final project for a Database course. I ran a Python script that used my Steam API key to fetch all of the items in my Counter-Strike 2 inventory and stored them in a .json file. I then parsed the data and converted it into an SQL insert script and populated the MariaDB tables. Using PHP, I made a web interface to display and edit the database of my skins. It includes the basic CRUD functions, as well as a few specific queries for the database that were required in the final project criteria.

PHP MariaDB mySQL

Steam Library Sorter

I made this as a little side project to learn Python over the summer of 2025. I've always wanted to be able to sort my Steam library by release date, but as far as I'm aware that is not an option, so I just wrote made a program to do it myself. The app runs a Python script that fetches the user's Steam library using their API key and User ID. After fetching, it saves all the data for faster access and allows the user to sort their library by newest or oldest games.

Game

Code Miss

This browser-based 3D game was developed in under 24 hours during the University of Mississippi's Hackathon 2025. It was rendered using A-Frame and programmed with JavaScript, using Git for version control. It also implements the DeepSeek LLM API for the NPCs. I was part of a 4-memeber team, and I was responsible for all of the 3D models as well as the main gameplay logic/scripting. The player's goal is to convince students to join the Coding Club within a limited time frame. Each NPC has a different personality, and the player must use a microphone to talk to convince them to join. The main location was modeled after the University's Computer Science building, Weir Hall.
